Classpublic class DVRManager
InheritanceDVRManager flash.events.EventDispatcher

Language Version : ActionScript 3.0
Product Version : FLVPlayback 2.5
Runtime Versions : Flash Player 10, AIR 1.5

Use the DVRManager class to manage queries issued to Flash Media Server for information about DVR streams. This class is optimized for use with the Flash Media Server DVRCast application available from Adobe or from Flash Video Streaming Service providers.
