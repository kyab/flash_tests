Language Version : ActionScript 3.0
Runtime Versions : AIR 1.0, Flash Player 9.0.28.0

The CuePointType class provides constant values for the type property on the info object of a MetadataEvent instance of type CUE_POINT. This type property is always EVENT, NAVIGATION, or ACTIONSCRIPT.

All of these constants can also be passed into the FLVPlayback.findCuePoint() and FLVPlayback.findNearestCuePoint() methods as the type parameter. The ALL and FLV constants describe multiple types of cue points for these methods.
